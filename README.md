# 前言

欢迎来到基于SSM（Spring、Spring MVC、MyBatis）的儿童影楼管理系统项目。该项目旨在为儿童摄影行业提供一个高效、便捷的管理系统，帮助影楼实现数字化、智能化管理。以下是对本项目的详细介绍。

## 内容介绍

本项目主要包括以下功能模块：客户管理、预约管理、拍摄管理、后期制作管理、财务管理等。通过这些模块，可以实现影楼的全方位管理，提高工作效率，降低人力成本。

客户管理模块：方便影楼录入客户信息，对客户进行分类管理，便于后续跟踪和维护。

预约管理模块：客户可以在线预约拍摄时间，影楼可以根据预约情况合理安排拍摄计划。

拍摄管理模块：记录拍摄过程中的相关信息，如拍摄主题、拍摄地点、摄影师等，便于后期制作和管理。

后期制作管理模块：对拍摄完成的照片进行编辑、修图，确保照片质量。

财务管理模块：统计各项目的收入、支出情况，为影楼提供财务数据支持。

## 技术介绍

本项目采用以下技术栈：

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12/14/16
